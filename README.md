<h1>COURSE: AI-Driven Software Development </h1><br/>
<p align="justify">This graduate-level course provides a comprehensive overview of using AI tools across each distinct stage of the Software Development Lifecycle (SDLC). The objective of 
the course is to help developers gain an understanding and idea of various AI tools and provisions that can be utilized to make software development practices effective,
efficient, and robust. AI must be integrated across the SDLC to meet the demands of today's fast-paced development world where system increments are released frequently.
<br/><br/>
<b>This course is now offered at Santa Clara University with course number CSEN 296B.</b> 
</p>
